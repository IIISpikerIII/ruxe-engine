<div class="titlenews"><h1> Как перенести свой дизайн в Ruxe Engine</h1></div>

         <div class="newsbody">
<p class="ro">
   <i>
      <b>1. Файловая структура темы оформления.</b>
   </i>
</p>
<p class="ro">Тема оформления должна иметь следующую файловую структуру:</p>
<ul>
    <li>/themes/</li>
    <ul>
        <li>/<b>название_темы</b>/</li>
        <ul>
             <li>commentform.html</li>
             <li>index.php</li>
             <li>list.dat</li>
             <li>message.html</li>
             <li>newsfullrecord.html</li>
             <li>newsrecord.html</li>
             <li>other.php</li>
             <li>users.php</li>
             <li>и другие файлы и каталоги на своё усмотрение</li>
        </ul>
    </ul>
</ul>
<p class="ro">Рассмотрим каждый элемент по отдельности.</p>
<center>
        <table class="optionstable" width="70%" cellpadding=1 cellspacing=0>
               <tr class="titletable"><td>Элемент</td><td>Описание</td></tr>
               <tr><td valign="top" align="center">название_темы</td><td>Папка с файлами темы оформления. Каждая папка в каталоге themes - новая тема. Имя папки соответствует её названию. В качестве имён рекомендуется использовать только строчные буквы латинского алфавита и цифры.</td></tr>
               <tr><td valign="top" align="center">commentform.html</td><td>Здесь находится оформление форм отправки комментариев и личных сообщений. Возьмите его из default темы оформления и вносите изменения, если необходимо.</td></tr>
               <tr><td valign="top" align="center">index.php</td><td>Шаблон оформления всех страниц сайта.</td></tr>
               <tr><td valign="top" align="center">list.dat</td><td>В данном файле прописываются имена пользовательских файлов, находящихся в папке темы, которые будут отображены в правой части раздела "Оформление" Админ-центра, в конце списка, и будут доступны для редактирования встроенным редактором.<br>
                                                                      Каждый новый файл - на новой строке.<br>
                                                                      Синтаксис файла следующий:<br>
                                                                      <i>имя_файла.расширение|описание_файла</i><br>
                                                                      <u><i>Пример:</i></u><br>
                                                                      <i>menu.html|Главное меню на сайте</i></td></tr>
               <tr><td valign="top" align="center">newsfullrecord.html</td><td>Здесь указывается стиль оформления <b>полной</b> записи новостного раздела. Рекомендуется за основу взять шаблон из default темы оформления.</td></tr>
               <tr><td valign="top" align="center">newsrecord.html</td><td>Здесь находится оформление <b>краткой</b> записи новостного раздела. Рекомендуется за основу взять шаблон из default темы оформления. Краткие записи отображаются в списке новостей.</td></tr>
               <tr><td valign="top" align="center">other.php</td><td>В файле указывается оформление различных модулей движка и стили их записей. С нуля создавать его нельзя - необходимо брать его из default темы оформления и вносить в него нужные изменения.</td></tr>
               <tr><td valign="top" align="center">users.php</td><td>Содержит в себе оформление всех пользовательских форм, меню и т.п. С нуля создавать его нельзя - необходимо брать его из default темы оформления и вносить в него нужные изменения.</td></tr>
        </table>
</center>
<p class="ro">В файлах с расширением .php допускается применять любые языки вёрстки (например, HTML) и веб-программирования, включая серверные (например, PHP). А в файлах с расширением .html - только клиентские языки веб-программирования (например, JavaScript, VisualBasic) и вёрстки. В указанных типах файлов используются HTML-теги и 
                <a href="http://engine.ruxesoft.net/documentation/commands.html">специальные команды движка</a>.</p>
